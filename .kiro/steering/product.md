# Saber-Translator

AI-powered manga/comic translation and editing tool with a web-based interface.

## Core Purpose
Provides end-to-end manga translation workflow: text detection → OCR → translation → inpainting → text rendering.

## Key Features
- Multi-engine OCR: MangaOCR (Japanese), PaddleOCR (multi-language), Baidu OCR, AI Vision OCR
- Multiple translation backends: SiliconFlow, DeepSeek, Volcano Engine, Google Gemini, Ollama, Sakura, custom OpenAI-compatible APIs
- LAMA-based intelligent text removal and background inpainting
- Manual annotation mode for text region editing
- Per-bubble editing with real-time preview
- Session management with auto-save
- Plugin system for extensibility
- Export to PNG, ZIP, PDF, CBZ formats

## Target Users
Manga translators and scanlation teams who need efficient batch translation with fine-grained control over output quality.

## Language
Primary UI and documentation is in Chinese (简体中文).
